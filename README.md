## Pré-requisitos Gerais

Para rodar qualquer um dos experimentos contidos nas pastas, você precisará ter instalado no seu sistema:

* **SWI-Prolog** (threaded, 64 bits, versão 10.0.2 ou superior)

Você pode verificar a instalação abrindo o terminal (PowerShell/CMD no Windows ou Bash no Linux) e digitando:
```bash
swipl
```
